### 1. Как я удаляла пустые строки из Повести Временных Лет: 

#### Работа проходила в приложении Sublime Text. Вначале я удалила пустые строки с помощью регулярного выражения: ^\n

![alt](https://github.com/MariaKsandrova/result.txt/blob/master/1.1.png)

#### Из текста не удалились все пустые строки, и я применила еще одно регулярное выражение: ^\s

### 2. Имена князей и городов, заканчивающихся на "слав" я искала через выражение: [А-Я][а-яѣ]+слав[а-яѣ]+ . Количество упоминаний 564.
